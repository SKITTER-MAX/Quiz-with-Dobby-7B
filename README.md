##  Repository
GitHub Repo: **https://github.com/SKITTER-MAX/Quiz-with-Dobby-7B**

Clone it:

```bash
git clone https://github.com/SKITTER-MAX/Quiz-with-Dobby-7B.git
cd Quiz-with-Dobby-7B

```

 Run Locally
Node.js Example
npm install
npm run dev


Open: http://localhost:3000

 How it Works

User starts quiz

Quiz engine requests question set from Dobby 7B

Model generates unique questions

UI displays questions + explanations

 Config Options
{
  "difficulty": "easy | medium | hard",
  "topics": ["Basics", "Smart Contracts", "NFTs", "DeFi", "Privacy"],
  "numQuestions": 10,
  "showHints": true
}
